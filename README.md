# GeoFill - 智能注册助手

根据地理位置智能生成注册信息并一键填写表单的浏览器插件。

## ✨ 功能特性

### 🌍 智能定位
- **自动检测 IP 位置** - 获取当前 IP 的国家、城市、州/省信息
- **直接使用 IP 城市** - 检测到的城市信息直接用于生成地址
- **手动刷新位置** - 点击刷新按钮重新检测当前 IP
- **手动切换国家** - 选择国家后自动重新生成匹配信息

### 📝 信息生成
- **真实地址关联** - 城市、州/省、邮编保持真实对应关系
- **精准电话格式** - 各国电话按真实格式和号段生成
- **本地化姓名** - 根据国家生成对应语言的姓名
- **多种用户名风格** - 基于姓名生成多种风格的用户名
- **强密码生成** - 12位包含大小写、数字、符号

### 🎨 界面功能
- **一键复制** - 点击复制按钮快速复制任意字段
- **深色/浅色主题** - 支持主题切换，保护眼睛
- **单独刷新** - 每个字段都可以单独重新生成
- **Toast 提示** - 操作成功后有提示反馈

### 📋 表单填写
- **增强字段检测** - 支持更多选择器和属性匹配
- **React/Vue 兼容** - 使用原生 setter 确保框架状态更新
- **全名字段支持** - 自动识别 fullname 字段并拆分填写
- **性别 radio 支持** - 支持 radio 按钮类型的性别选择
- **密码双填** - 自动填写密码和确认密码

## 📦 支持的信息类型

| 字段 | 说明 |
|------|------|
| 名/姓 | 根据国家生成对应语言的姓名 |
| 性别 | 随机生成男/女 |
| 生日 | 随机生成 1970 年后的生日（18-55岁）|
| 用户名 | 基于姓名生成多种风格的用户名 |
| 邮箱 | 用户名 + 可选邮箱后缀 |
| 密码 | 12位强密码（包含大小写、数字、符号）|
| 电话 | 各国真实号段和格式 |
| 地址 | 各国本地化街道名称 |
| 城市 | IP 检测的城市或随机城市 |
| 州/省 | 与城市正确对应 |
| 邮编 | 基于州的真实前缀 |
| 国家 | 可手动选择，自动更新相关信息 |

## 🚀 安装方法

### Chrome / Edge

1. 打开扩展程序页面
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目文件夹

### Firefox

1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击「临时加载附加组件」
3. 选择项目中的 `manifest.json` 文件

## 📖 使用方法

1. 打开任意注册页面
2. 点击浏览器工具栏中的插件图标
3. 查看自动生成的信息（可点击 🔄 刷新 IP 位置）
4. 可手动修改任意字段或切换国家
5. 点击「填写表单」按钮
